# 文殊坊铺位图块模型

打开 `index.html` 可查看互动铺位图。页面支持：

- 按分析项切换：面积、租约到期、业态、月坪效、月租金、租销比、人流量。
- 按楼层切换：B1、1F、2F、3F。
- 点击色块查看分类、来源图层、像素位置，并显示局部放大图。
- 在左侧搜索图块编号或分类。

模型数据文件位于 `data/tile_model.json`，网页使用 `data/maps.js` 加载同一份数据。

说明：本版本基于 PNG 图片自动提取色块区域。租户名称、面积、租金、销售额等字段仍以原图文字为准；如后续提供 Excel、CAD、SVG 或原始矢量图，可把这些字段精确写入每个图块。
